# SurveyMine Pro v7.0 ⚒️
### Sistema de Levantamiento Subterráneo y Geodesia Minera

SurveyMine Pro es una **Progressive Web App (PWA)** diseñada específicamente para el trabajo de campo en minería subterránea y túneles. Permite realizar levantamientos topográficos rápidos directamente desde un dispositivo móvil, optimizando la toma de datos en entornos sin conectividad.

## 🚀 Características Principales
- **Funcionamiento 100% Offline:** Gracias a la tecnología Service Worker, la App funciona sin señal de internet dentro de las labores mineras.
- **Cálculo de Coordenadas UTM:** Procesa Azimut, Inclinación y Distancia en tiempo real para generar coordenadas (E, N, Z).
- **Auto-incremento Inteligente:** Sistema de nomenclatura automática (P01, P02...) para no perder el ritmo del levantamiento.
- **Visualización Dinámica:** Croquis interactivo con soporte para zoom táctil y gestos.
- **Exportación Profesional:** Genera archivos Excel (.xlsx) con formato técnico listo para post-procesamiento en CAD/GIS.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Diseño responsivo para móviles)
- JavaScript Vanilla (Lógica trigonométrica y procesamiento)
- SheetJS (Motor de exportación de datos)
- PWA Standards (Manifiesto y Service Worker)

## 📍 Aplicación en Campo
Desarrollado con enfoque en las condiciones técnicas de la minería en **Zaruma, Ecuador**, cumpliendo con estándares de precisión para poligonales subterráneas.
